# ◆ Abe — ABF Deal Screen

**Asset-based finance dual-track deal screening platform by i80 Group.**

Abe runs a collateral analytics track (Track A) and a corporate/platform underwriting track (Track B) in parallel, then combines them into an integrated deal screen with red flags, contradiction analysis, and diligence questions.

---

## Quick start

```bash
# 1. Install dependencies
pip install streamlit pandas openpyxl

# 2. Generate sample deal data (optional — for demo)
python create_fixtures.py

# 3. Launch the app
streamlit run app.py
```

Opens in your browser at **http://localhost:8501**.

## Two ways to run

### Option A: Streamlit app (recommended)

```bash
streamlit run app.py
```

Visual 10-step wizard. Upload files, fill forms, review red flags, get a deal screen.

### Option B: CLI pipeline

```bash
python create_fixtures.py   # generate sample data
python run_pipeline.py       # run full pipeline, outputs to /outputs
```

Runs the analytics backend directly. No UI. Outputs markdown + JSON to the outputs folder.

---

## Using with real deal data

1. **Loan tape:** Export from borrower's LMS as `.xlsx` or `.csv`. One row per loan. Abe maps columns automatically.

2. **Corporate financials:** Open `templates/corporate_financials.xlsx`. Fill in the Income Statement and Balance Sheet tabs from the borrower's audited/reviewed statements. Minimum 2 years.

3. **Funding facilities:** Open `templates/funding_facilities.xlsx`. One row per existing facility.

4. **Platform questionnaire:** Fill in `templates/platform_questionnaire.json` based on diligence materials and borrower calls.

5. Run the app, upload everything, click through the steps.

## What Abe does NOT do

- Make investment decisions (human-only)
- Parse unstructured PDFs (V1 feature — use the Excel templates for now)
- Replace analyst judgment (it augments, humans review everything)
- Calculate anything with the LLM (all math is deterministic Python)

---

## Stack

- **Python** — all analytics
- **Streamlit** — UI
- **pandas / openpyxl** — data processing
- **Claude API** — field mapping, red flags, scorecard, screen narrative (optional — works without it using deterministic fallbacks)
